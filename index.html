Pixelator

By Isabelle Lee

I. Description

Pixelator turns images into arrays of integers that represent pixel data. There are four
numbers for each pixel denoting red, green, blue, and alpha (opacity) values that
range from 0 to 255.

I chose a set of 200 pixels by 200 pixels images that the user could choose from.
Because this would yield a total of 40,000 pixels and 4 data points for each, which
would result in an array with 160,000 elements, I chose to average the values of
100 pixel regions after separating them into their respective red, green, blue, and
alpha arrays. This resulted in 4 arrays of length 400 each.

I then fed these arrays into different oscillators and filters to produce and
